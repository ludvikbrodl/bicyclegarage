GIT WALKTRHOUGH:
ni bör göra en pull varje gång innan ni börjar så att ni får den nyaste versionen av koden.

*to pull from github
1) TERMINALEN:
cd to your workspace
2) TERMINALEN:
git pull

nu laddas allt ner till er workspace, glöm inte att refresha i eclipse om den var öppen under pullen.

*to make a commit:
1) TERMINALEN:
cd to your workspace
2) TERMINALEN:
git commit -a

Nu öppnas en text editor, där skriver du vad du har ändrat och vad som har gjorts, sen sparar du och stänger ner.

3) TERMINALEN:
git push origin master

om det ger success så hurra! Om det står att den failar så gör ni en pull (se nedan)
